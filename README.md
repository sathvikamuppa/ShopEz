ShopEZ

ShopEZ is a MERN Stack E-Commerce Application developed as part of the Full Stack Developer MERN Stack Internship. The application provides a seamless online shopping experience with user authentication, product browsing, category filtering, shopping cart functionality, and order management.

Features

* User Authentication
* Product Catalog
* Category Filtering
* Product Search
* Shopping Cart Management
* Order Management
* MongoDB Database Integration
* Responsive User Interface
* Admin Product Management

Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* JavaScript
* HTML
* CSS
